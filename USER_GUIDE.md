# WayneMemo 新手指引

欢迎使用 WayneMemo！这是一个完全离线的本地笔记应用，所有数据存储在您的电脑上，保护您的隐私。

---

## 📥 安装与启动

### 首次安装

1. **下载安装包**
   - macOS Apple Silicon: `WayneMemo-1.0.0-arm64.dmg`
   - macOS Intel: `WayneMemo-1.0.0.dmg`
   - Windows: `WayneMemo Setup 1.0.0.exe`

2. **安装应用**
   - **macOS**: 打开 `.dmg` 文件，将 WayneMemo 拖到应用程序文件夹
   - **Windows**: 运行 `.exe` 安装程序，按提示完成安装

3. **启动应用**
   - **macOS**: 从启动台或应用程序文件夹启动
   - **Windows**: 从开始菜单启动

---

## 🚀 快速开始

### 第一次使用

1. **启动应用后**，您会看到一个空白的编辑界面
2. **点击标题区域**，输入笔记标题（如："我的第一篇笔记"）
3. **在下方编辑区域**，开始输入内容
4. **自动保存**：内容会自动保存，无需手动操作

### 基本操作

#### 创建新笔记
- 点击右侧工具栏的 **"笔记列表"** 图标（第一个）
- 点击 **"+ 新建笔记"** 按钮
- 输入标题和内容

#### 查看笔记列表
- 按 `Cmd/Ctrl + S` 或点击工具栏 **"笔记列表"** 图标
- 显示所有笔记，按修改时间排序

#### 搜索笔记
- 按 `Cmd/Ctrl + F` 或点击工具栏 **"搜索"** 图标
- 输入关键词，实时显示匹配结果

#### 归档笔记
- 在笔记列表中，点击笔记右侧的 **"归档"** 图标
- 归档的笔记不会显示在主列表中
- 按 `Cmd/Ctrl + U` 查看归档笔记

## 🌓 便签模式

WayneMemo 支持便签模式，适合快速记录。

### 如何触发便签模式

**自动触发**：将窗口调整到小尺寸
- 宽度 < 500px **或** 高度 < 700px
- 推荐尺寸：**400 x 600**

**效果**：
- 工具栏自动隐藏
- 内容区域占满全屏
- 适合快速记录和便签使用

### 退出便签模式

**自动退出**：将窗口放大
- 宽度 ≥ 500px **且** 高度 ≥ 700px
- 推荐尺寸：**800 x 800** 或更大

**效果**：
- 工具栏自动显示
- 恢复标准布局

## 🔌 插件功能

### 内置插件

#### 1. 链接预览
**功能**：自动识别 URL 并显示预览

**使用方法**：
- 在笔记中输入网址（如 `https://www.baidu.com` 或 `baidu.com`）
- 自动显示链接预览卡片

**支持格式**：
- 完整 URL：`https://example.com`
- 简短域名：`example.com`
- 带路径：`example.com/path`

#### 2. 计算器
**功能**：自动识别数学表达式并计算结果

**使用方法**：
- 在笔记中输入数学表达式
- 自动显示计算结果

**支持运算**：
- 基础运算：`2 + 3 * 4`
- 括号：`(2 + 3) * 4`
- 小数：`3.14 * 2`
- 复杂表达式：`(100 - 20) / 4 + 5`

### 插件管理

1. 按 `Cmd/Ctrl + E` 打开插件管理
2. 查看已安装的插件
3. 启用/禁用插件

## ⚡ 快捷码 (Snippets)

快捷码可以帮助您快速插入常用文本。

### 使用快捷码

1. 在编辑区输入 `/`
2. 输入快捷码关键词
3. 选择匹配的快捷码
4. 自动插入对应文本

### 管理快捷码

1. 点击工具栏 **"快捷码"** 图标
2. **添加快捷码**：
   - 输入触发词（如 `email`）
   - 输入替换文本（如 `your@email.com`）
   - 点击 **"添加"**
3. **删除快捷码**：点击快捷码右侧的删除图标

### 快捷码示例

| 触发词 | 替换文本 | 用途 |
|--------|---------|------|
| `email` | `your@email.com` | 快速插入邮箱 |
| `phone` | `123-456-7890` | 快速插入电话 |
| `addr` | `Your Address` | 快速插入地址 |
| `sign` | `Best regards,\nYour Name` | 快速插入签名 |

## 🕐 操作历史

WayneMemo 会自动记录您的编辑历史。

### 查看历史

1. 按 `Cmd/Ctrl + Shift + H` 打开操作历史
2. 查看该笔记的所有历史版本
3. 显示操作时间和内容快照

### 撤销到历史版本

1. 在操作历史中找到目标版本
2. 点击 **"回滚"** 按钮
3. 笔记内容恢复到该版本

**注意**：回滚操作会创建新的历史记录，不会删除原有历史。

## 🔔 每日碎片笔记提醒

WayneMemo 会在每天 20:00 检查您的碎片笔记（少于 50 字符的笔记），并发送系统通知提醒您完善。

### 功能说明

- **自动检测**：每天 20:00 自动检查
- **碎片笔记定义**：内容少于 50 字符的笔记
- **系统通知**：显示碎片笔记数量
- **快速跳转**：点击通知，自动高亮显示碎片笔记

### 手动触发

在设置页面，点击 **"立即检查碎片笔记"** 按钮。

## 💾 数据管理

### 数据存储位置

您的笔记数据保存在本地：

- **macOS**: `~/Documents/WayneMemo_Data/`
- **Windows**: `C:\Users\<用户名>\Documents\WayneMemo_Data\`

### 打开数据文件夹

1. 按 `Cmd/Ctrl + ,` 打开设置
2. 找到 **"数据存储位置"** 部分
3. **点击蓝色路径文本**
4. 文件管理器自动打开数据文件夹

### 数据文件结构

```
WayneMemo_Data/
├── metadata.json          # 笔记索引和配置
├── notes/                 # 笔记内容
│   ├── note_xxxxx.md     # Markdown 格式笔记
│   └── ...
├── archived/              # 归档笔记
│   └── ...
└── logs/                  # 操作历史
    └── ...
```

### 数据备份

**推荐方式**：
1. 打开数据文件夹（见上方）
2. 复制整个 `WayneMemo_Data` 文件夹
3. 保存到备份位置（如云盘、移动硬盘）

**恢复数据**：
1. 关闭 WayneMemo
2. 将备份的 `WayneMemo_Data` 文件夹复制回原位置
3. 重新启动 WayneMemo

## 🔧 常见问题

### Q: 如何删除笔记？
**A**: 目前支持归档笔记（`Cmd/Ctrl + U`），归档后的笔记不会显示在主列表中。如需完全删除，可以在数据文件夹中手动删除对应的 `.md` 文件。

### Q: 笔记内容没有自动保存？
**A**: WayneMemo 会在编辑停止 5 秒后自动保存。如果遇到问题，请检查数据文件夹是否有写入权限。

### Q: 如何导出笔记？
**A**: 笔记以 Markdown 格式存储在数据文件夹中，可以直接复制 `.md` 文件到其他位置或导入其他 Markdown 编辑器。

### Q: 全局快捷键 Cmd/Ctrl + . 不工作？
**A**:
1. 确保没有其他应用占用该快捷键
2. 重启 WayneMemo
3. 检查系统权限设置（macOS 需要辅助功能权限）

### Q: 便签模式无法退出？
**A**: 将窗口放大到 500px 宽 **且** 700px 高以上，工具栏会自动显示。推荐尺寸：800 x 800 或更大。

### Q: 如何卸载 WayneMemo？
**A**:
- **macOS**: 从应用程序文件夹删除 WayneMemo
- **Windows**: 从控制面板卸载程序
- **数据清理**: 手动删除 `WayneMemo_Data` 文件夹（如需保留数据请先备份）

## 💡 使用技巧

### 1. 快速记录
- 使用全局快捷键 `Cmd/Ctrl + .` 快速唤出 WayneMemo
- 切换到便签模式（400x600），专注于内容输入
- 编辑完成后按 `Cmd/Ctrl + .` 隐藏窗口

### 2. 高效搜索
- 使用 `Cmd/Ctrl + F` 快速搜索
- 支持搜索标题和内容
- 实时显示匹配结果

### 3. 笔记分类
- 使用标签：在标题中添加 `#标签名`
- 使用前缀：如 `[工作] 项目计划`
- 使用归档：完成的笔记归档，保持列表整洁

### 4. Markdown 技巧
- **标题**: `# 一级标题`, `## 二级标题`
- **列表**: `- 项目` 或 `1. 项目`
- **链接**: `[文本](URL)`
- **代码**: `` `代码` `` 或 ``` 代码块 ```
- **加粗**: `**加粗文本**`
- **斜体**: `*斜体文本*`

### 5. 快捷码妙用
- 设置常用文本快捷码
- 设置模板快捷码（如会议记录模板）
- 使用 `/` 快速触发
